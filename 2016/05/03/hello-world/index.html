把博客从jekyll换成了hexo,之前的建好了一年也就写了三篇文章，两篇还是关于jekyll配置的内容，所以就不做迁移了（其实是懒）。

hexo

hexo设计的很容易上手，主要是源码和发布的内容托管分开了。源码用了oschina的git私有仓管理，托管还是在github上。

next

主题用了next，集成了很多第三方模块。暂时配置了多说、谷歌统计、swiftype。

nodejs

这两天刚发布了nodejs v6.0 ,mac上直接用6.0 版本的安装和运行都会有问题，所以还是用了和pc端一样的v4.4.3。
node的npm源还是被qiang的很厉害的，在windows 上也不想直接用淘宝的cnpm ，所以和mac 上一样，用了 nrm 这个包来管理版本。

nrm

  1. 全局安装nrm
    npm install -g nrm --registry=https://registry.npm.taobao.org
  2. 查看包含的源
    nrm ls
  3. 切换成淘宝的源
    nrm use taobao

nvm

mac用了nvm做node的版本切换，windows上暂时还没折腾。

markdown

markdown的编辑器还是直接用了sublime Text 3 加插件MarkdownEditor和一个预览的 OmniMarkupPreviewer ，mac上应该可以直接用mou了，现装下。
